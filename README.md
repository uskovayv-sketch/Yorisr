# Legal Translator Bot

Юридический помощник на базе YandexGPT

## Возможности
- Перевод юридических текстов на понятный язык
- Объяснение юридических терминов

## Команды
- /start - начать работу
- /help - помощь
- /term [термин] - объяснить термин

## Технологии
- Python
- Aiogram 3.x
- YandexGPT API

## Переменные окружения (обязательно!)
- `TELEGRAM_TOKEN` - токен бота от @BotFather
- `YANDEX_API_KEY` - API-ключ YandexGPT
- `YANDEX_FOLDER_ID` - ID каталога в Yandex Cloud
